## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Faey2023/fintech-landing

   ```

2. **Navigate to the project directory:**

   ```bash
   cd fintech-landing
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Run the project:**
   ```bash
   npm run dev
   ```
